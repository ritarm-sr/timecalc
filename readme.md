# timecalc
経過時間(時刻ではない)の加減算を行います。

## インストール
1. [ここ](https://github.com/ritarm-sr/timecalc/releases/latest) から`timecalc.exe`をダウンロード
2. 適当なフォルダ2保存
3. windowsボタン
4. 🔍️`システム環境変数の編集`
5. `環境変数(N)`
6. 上の表の`Path`を選択し`編集`
7. `新規(N)` => 保存したフォルダーのアドレスを入力し`OK`
8. `OK` => `OK` ウィンドウが全部閉じたらOK

## 使い方
cmd または pwsh で >`timecalc` に続いて以下コマンドを入力

## 記法(コマンド)
加算は `+` または `p`
減算は `-` または `m`
乗算は `*`
除算は `/`
括弧は `(` `)`
60進法は `HH:MM`
10進法は `H.M`

`*` と `/` の後ろは整数または小数のみ使えます。`HH:MM` や括弧付き式は使えません。

例
```powershell
# 100:58 + 25:45 - 12:22
> timecalc 100:58 p 25:45 m 12:22
#結果 60進法 (10進法)
114:21 (114.350)
```

```powershell
# 60進法と10進法の混合
> timecalc 49:19 p 61.25
#結果 60進法 (10進法)
110:34 (110.567)
```

```powershell
# 括弧と四則演算
> timecalc "(1:30 + 0:30) * 2 - 0:15"
#結果 60進法 (10進法)
3:45 (3.750)
```

```powershell
# * / の右辺は数値のみ
> timecalc "1:00 * 1.5"
#結果 60進法 (10進法)
1:30 (1.500)
```
