
# timecalc

経過時間の加減乗除・括弧計算をコマンドラインで手軽に行えるツールです。

---

## インストール

PowerShellで下記コマンドを実行するだけで導入できます（管理者権限不要）。

```
powershell -ep bypass -c "irm https://raw.githubusercontent.com/ritarm-sr/timecalc/main/install.ps1 | iex"
```

インストール後は `timecalc` または `tc` でどこからでも呼び出せます。

---

## 使い方

コマンドプロンプトやPowerShellで、

```powershell
timecalc [式]
```

または

```powershell
tc [式]
```

と入力してください。

---

## 入力例・記法

- 加算: `+` または `p`
- 減算: `-` または `m`
- 乗算: `*`
- 除算: `/`
- 括弧: `(` `)`
- 60進法: `HH:MM`（例: 1:30）
- 10進法: `H.M`（例: 1.5）

※ `*` と `/` の右辺は数値のみ（`1.5` など）。

### サンプル

```powershell
# 100:58 + 25:45 - 12:22
tc 100:58 p 25:45 m 12:22
# → 114:21 (114.350)

# 60進法と10進法の混合
tc 49:19 p 61.25
# → 110:34 (110.567)

# 括弧と四則演算
tc "(1:30 + 0:30) * 2 - 0:15"
# → 3:45 (3.750)

# * / の右辺は数値のみ
tc "1:00 * 1.5"
# → 1:30 (1.500)
```

```powershell
# → 3:45 (3.750)

# p, m, t, d のみで構成される式はクォート不要
tc 1:00 t 2 d 2
# → 1:00 (1.000)

# * / の右辺は数値のみ
tc 1:00 t 1.5
# → 1:30 (1.500)
```
