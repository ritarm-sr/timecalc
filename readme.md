# timecalc

経過時間の加減乗除・括弧計算をコマンドラインで手軽に行えるツールです。

---

## インストール

PowerShellで下記コマンドを実行するだけで導入できます（管理者権限不要）。

```
powershell -ep bypass -c "irm https://raw.githubusercontent.com/ritarm-sr/timecalc/main/install.ps1 | iex"
```

インストール後は `timecalc` または `tc` でどこからでも呼び出せます。

---

## 使い方

コマンドプロンプトやPowerShellで、

```
timecalc [式]
```

または

```
tc [式]
```

と入力してください。

---

## 入力例・記法

- 加算: `+` または `p` (plus の頭文字)
- 減算: `-` または `m` (minus の頭文字)
- 乗算: `*` または `t` (times の頭文字)
- 除算: `/` または `d` (division の頭文字)
- 括弧: `(` `)`
- 60進法: `HH:MM`（例: 1:30）
- 10進法: `H.M`（例: 1.5）

※ 乗算 `*` `t` および 除算 `/` `d` の右辺は数値のみ（`1.5` `4` など）。


### サンプル

```powershell
# 100時間58分に25時間45分を加え、さらに12時間22分を減らす
# 100:58 + 25:45 - 12:22
tc 100:58 p 25:45 m 12:22
# → 114:21 (114.350)

# 60進法と10進法の混合
# 49時間19分に61.25時間を加える
tc 49:19 p 61.25
# → 110:34 (110.567)

# 括弧や演算子を混ぜる場合、スペース区切りで "" が必要なこともあります
# 1時間30分に30分を加えたものを2倍して、15分減らす
tc "(1:30 + 0:30) t 2 m 0:15"
# → 3:45 (3.750)

# p, m, t, d のみで構成される式はクォート不要
# 1時間を2倍して、その半分の時間を求める
tc 1:00 t 2 d 2
# → 1:00 (1.000)

# * / の右辺は数値のみ
# 1時間を1.5倍する
tc 1:00 t 1.5
# → 1:30 (1.500)
```
